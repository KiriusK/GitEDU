# Подсказка по Git

### Создание репозитория
```sh
git init
```
### Проверка состояния репозитория
```sh
git status
```
### Добавить файл в индекс для коммита
```sh
git add file_name
```

```sh
git restore file_name
```

```sh
git restore --staged file_name
```

```sh
git commit -m "Message for commit"
```

```sh
git log
```

```sh
git log --oneline
```

```sh
git checkout hash_number
```

```sh
git checkout master
```

```sh
git diff
```

```sh
git dif hash_number
```
